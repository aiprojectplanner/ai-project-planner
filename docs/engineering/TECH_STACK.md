# Technology Stack 
 
 ## Frontend 
 - **Framework**: React 18 
 - **Build Tool**: Vite 
 - **Styling**: Tailwind CSS 
 - **Icons**: Lucide React 
 - **Gantt Library**: Custom lightweight Gantt implementation 
 
 ## State Management 
 - **Zustand**: Modular stores for authentication (`authStore.js`) and project data (`projectStore.js`). 
 
 ## Backend 
 - **Compute**: Vercel Serverless Functions (Node.js) 
 - **AI API**: OpenRouter (Unified access to LLMs like Gemini 2.5 Flash) 
 
 ## Database & Auth (BaaS) 
 - **Platform**: Supabase 
 - **Database**: PostgreSQL 
 - **Authentication**: Supabase Auth (JWT based) 
 
 ## Hosting & Deployment 
 - **Frontend & API**: Vercel 
 - **Environment Management**: `.env.local` (local) / Vercel Environment Variables (production)

 - **Platform**: GitHub
